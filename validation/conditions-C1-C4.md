# Empirical Validation Conditions — C1 to C4

These four conditions are independently falsifiable.  
C1 and C2 are the primary Phase 1 targets for empirical work.

---

## C1 — Memory Accumulation

**Claim:** Agent performance improves measurably as episodic  
memory accumulates across sessions.

**Method:** Random Forest classification with null-memory baseline.  
**Evaluator:** Blind panel — output quality scoring.  
**Status:** 🔄 Protocol defined — awaiting empirical run

## C2 — Memory Loss

**Method:** Controlled ablation of memory layers M1/M2.

**Status:** 🔄 Protocol defined — awaiting empirical run
